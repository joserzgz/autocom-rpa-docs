autocom.rpa.encript.decrypt_str(value, keystr=b'BnBj9a8Zgs-D3HG1vlZGNTyUGKzyJXSStIGY6RT5rcU=')

Desencripta una cadena y la regresa sin encriptación. Deberá proporcionarse el argumento «keystr» con la cadena secreta que fue utilizada en su encriptación.

Args:

  • value (str): Cadena encriptada.

  • keystr (bytes, optional): Cadena secreta utilizada en la encriptación. , la cadena secreta se envía con formato base64 a partir de un número de 32 bytes. Defaults to b’BnBj9a8Zgs-D3HG1vlZGNTyUGKzyJXSStIGY6RT5rcU=”.

Returns:

  • str: Regresa la cadena sin encriptación.

autocom.rpa.encript.encrypt_str(value, keystr=b'BnBj9a8Zgs-D3HG1vlZGNTyUGKzyJXSStIGY6RT5rcU=')

Encripta una cadena y la regresa encriptada. Garantiza que la cadena encriptada no sea vulnerada sin el argumento «keystr».

Args:

  • value (str): Cadena a encriptar.

  • keystr (bytes, optional): Cadena secreta que sirve para asegurar la encriptación, la cadena secreta se envía con formato base64 a partir de un número de 32 bytes. Defaults to b’BnBj9a8Zgs-D3HG1vlZGNTyUGKzyJXSStIGY6RT5rcU=”.

Returns:

  • str: Regresa la cadena encriptada.
