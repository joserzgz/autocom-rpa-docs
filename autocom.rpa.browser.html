autocom.rpa.browser.check_connection(host, port=80, timeout=3)

Comprueba si hay conexión hacia el host. El host puede ser una url o dirección pública o en una intranet.

Args:

  • host (string): Url o IP destino de la prueba de conexión.

  • port (int, optional): Puerto de conexión a utilizar. Defaults to 80.

  • timeout (int, optional): Tiempo de espera en segundos antes de determinar la conexión fallida. Defaults to 3.

Returns:

  • boolean: Regresa un valor que indica si se logró establecer la conexión.

autocom.rpa.browser.wait_for_connection(host, port=80, totaltimeout=30)

Espera hasta que se establezca la conexión al host.

Args:

  • host (string): Url o IP destino de la conexión.

  • port (int, optional): Puerto de conexión. Defaults to 80.

  • totaltimeout (int, optional): Tiempo de espera en segundos para esperar la conexión. Defaults to 30.
