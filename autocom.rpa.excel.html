class autocom.rpa.excel.ExcelBook

Bases: object

Clase que maneja la conexión habia un libro de excel, ya sea nuevo (creado con el mismo objecto) o abriendo un libre existente. Mediante el objeto instanciado de esta clase se controla el libro mediante los métodos incluidos en la clase.

Returns:

  • ExcelBook: Clase para el manejo y control de un libre de excel.

autoFitColumns(max_width=None)

Ajusta el ancho de todas las columna con datos en la hoja vigente.

Args:

  • max_width (int | None, optional): En caso de recibir un valor tipo int ese valor se considera como el valor máximo del ancho de columna, en caso de recibir un valor None no se establece un ancho máximo. Defaults to None.

fill_from_dataset(rows, columns=None, sheet_name=None)

Llena una hoja de excel comenzando desde A1 con los renglones recibidos en el argumento rows.

Args:

  • rows (arreglo de JSON): Se recibe un arreglo de objetos JSON, cada objeto corresponde a un renglón de la hoja de excel.

  • columns (arreglo de JSON, optional): Se recibe un arreglo de objetos JSON con la definición de cada columna que integrará el llenado de la hoja. Defaults to None.

    Los campos que podrá contener cada objeto columna son:

    {
    "name": "<str>",
    "title": "<str>",
    "width": "auto" | <int>
}

    • name: Nombre del campo del registro JSON correspondiente al renglón.

    • title: Título a utilizar en el encabezado de la columna.

    • width: Ancho de la columna, si se envía una cadena «auto» se calculará el ancho automáticamente, o puede enviarse un valor tipo int que indica el ancho de la columna.

  • sheet_name (str | None, optional): Nombre de la hoja donde se llenará la rejilla, si se envía None se utilizará la hoja vigente. Defaults to None.
